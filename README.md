# ProjectDicot

##### เป็นเกม 3D ที่ผู้เล่นจะได้รับบทเป็นช่างไฟคนหนึ่ง ที่ได้รับหน้าที่มาฟื้นฟูและซ่อมแซมให้โรงงานแห่งนี้ กลับมาน้ำไหล ไฟสว่างอีกครั้ง ซึ่งระบบท่อน้ำหลักจะแตกแขนงไปตามทางต่าง ๆ ผู้เล่นจะต้องค้นหาและซ่อมแซมให้มันกลับมาทำงานได้อีกครั้ง 

##### Concept/Theme ของเกมคือ Root 

##### เกมนี้เป็นเกมที่ทำระหว่างเรียนอยู่ชั้นปีที่ 3 มหาวิทยาลัยเชียงใหม่ อาจมีบัคและข้อผิดพลาดที่เกิดขึ้นภายในเกม
